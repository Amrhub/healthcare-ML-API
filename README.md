# healthcare-ML-API

## API Documentation

[API Documentation](https://healthcaregp.herokuapp.com/docs)

URL: https://healthcaregp.herokuapp.com/<br>
endpoint: "/data": <br>

## <b>_POST_<b> request [REQUIRED] format:

```js
{
  id: float;
  heartRate: float;
  spo2: float;
  temp: float;
  ECG: List / Array;
  GPS: List / Array;
}
```
